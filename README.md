#Census Atlas Japan

The data used to create these atlases are available from [Statistics Japan](http://www.stat.go.jp/english/), however, for this project these were provided to me by [Keiji Yano](http://www.ritsumei.ac.jp/~yano/profile/index_en.html) and [Tomoki Nakaya](http://www.ritsumei.ac.jp/~nakaya/) who also supplied the boundaries and various lookups. The places highlighted on the map were extracted from the Japan gazetteer file from Geonames; and code used to create the maps is [here](http://rpubs.com/alexsingleton/japan).

Some of these files are large, you might be best right clicking and selecting download rather than just clicking them.
